# Screenshot and Image Assets

## Structure

- `screenshot-1.png`, `screenshot-2.png` - General documentation screenshots for README.md
- `chrome-webstore/` - Chrome Web Store specific assets for the extension page
  - `screenshot-1-1280x800.png` … `screenshot-5-1280x800.png`
  - `promo-440x280.png`
  - `promo-1280x800.png`
  - `promo-1400x560.png`

## README.md Requirements

| Title            | Content                             |
| ---------------- | ----------------------------------- |
| `screenshot-1.png` |  popup screenshot  |
| `screenshot-2.png` |  feature screenshot  |

## Chrome Web Store Requirements

### Screenshots (Required)

- **Format**: PNG, JPG, GIF
- **Size**: 1280x800px (16:10)
- **Count**: 1–5 recommended
- **Content**: Show key features of the extension
- **Filenames**: `screenshot-1-1280x800.png` … `screenshot-5-1280x800.png`

### Small Promo Tile (Recommended)

- **Size**: 440x280px (11:7)
- **Purpose**: Shown in recommendation sections and category listings
- **Filename**: `promo-440x280.png`

### Promo Image (Optional)

- **Size**: 1280x800px (5:2)
- **Filename**: `promo-1280x800.png`

### Promo Image (Optional)

- **Size**: 1400x560px (5:2)
- **Purpose**: Promotional campaigns in the store
- **Filename**: `promo-1400x560.png`
